# Future Work

This project is currently under active development as part of the ACA IIT Kanpur Database Track. The following enhancements are planned for future iterations.

## Hospital Management System

* Extend the database schema to include billing and payment modules.
* Add room allocation and admission management functionalities.
* Implement medical records and prescription management.
* Develop advanced analytical queries using window functions and Common Table Expressions (CTEs).
* Optimize frequently executed queries through indexing strategies.
* Implement transaction management to ensure ACID compliance.

## Quick Commerce System

* Introduce inventory and warehouse management modules.
* Add payment processing and refund handling mechanisms.
* Implement delivery partner and order tracking functionalities.
* Develop customer analytics and revenue analysis queries.
* Incorporate transaction handling for order placement and payment workflows.
* Perform query optimization for large-scale transactional datasets.

## General Enhancements

* Add stored procedures and triggers.
* Implement database normalization and performance benchmarking.
* Analyze query execution plans and optimize bottlenecks.
* Expand sample datasets to simulate real-world scenarios.
* Improve project documentation and ER diagrams.

## Long-Term Goals

* Integrate the database systems with a front-end application.
* Develop dashboards for data visualization and business insights.
* Deploy the systems on a cloud-based database platform.
