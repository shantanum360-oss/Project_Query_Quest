# Query Quest

A collection of advanced SQL database projects focused on designing robust relational database systems and performing high-performance query optimization. This repository demonstrates the application of advanced SQL concepts, transaction management, and database optimization techniques in real-world scenarios.

## 📌 Project Overview

Query Quest is a database engineering project developed under the Association for Computing Activities (ACA), IIT Kanpur. The project involves designing and implementing relational database systems for multiple domains while ensuring efficiency, scalability, and data consistency.

The repository currently includes:

- 🏥 Hospital Management System Database
- 🛒 Quick Commerce Application Database

## 🎯 Objectives

- Design normalized relational database schemas for real-world applications.
- Implement efficient data retrieval and manipulation techniques.
- Ensure data integrity and consistency using transaction management.
- Optimize query performance for large-scale transactional workloads.

## ⚙️ Features

### Database Design
- Designed comprehensive relational schemas for:
  - Hospital Management System
  - Quick Commerce Application
- Established relationships using primary and foreign keys.
- Applied normalization principles to reduce redundancy.

### Advanced SQL Operations
- Complex `JOIN` operations
- Aggregate functions for analytical insights
- Window functions for advanced data analysis
- Common Table Expressions (CTEs)
- Nested queries and subqueries

### Transaction Management
- Enforced **ACID properties**:
  - Atomicity
  - Consistency
  - Isolation
  - Durability
- Implemented strict transaction handling mechanisms.

### Query Optimization
- Optimized SQL queries for faster execution.
- Reduced query execution time under heavy transactional workloads.
- Minimized bottlenecks in large-scale database operations.

## 🛠️ Technologies Used

- SQL
- Relational Database Management Systems (MySQL/PostgreSQL)
- Database Normalization Techniques
- Transaction Management
- Query Optimization Techniques
